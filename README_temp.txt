### mrp-v06.9 ###############################################################################

### mrp-v06.8 ###############################################################################
2009/12/4
mrp-v06.7からの変更点(mrp-v06.7 -> mrp-v06.8)
  a) Microsoft C Compiler(VC++)，Intel C/C++ Compiler, GCC に対応
  b) pm_accuracy(予測値の量子化精度) = "-1" への対応を停止

  b) その他ソースコードの整理

### mrp-v06.7 ###############################################################################
2009/10/8
mrp-v06.6からの変更点(mrp-v06.6 -> mrp-v06.7)
  a) 予測次数及び特徴量Uの算出に用いる参照画素数(NUM_UPELS)をともに"1"に設定した場合に，
     参照画素配置が適切に設定されないバグを修正.
     (通常は予測次数が2次以上ないと予測係数を修正できないため1次に設定することはない.)
  b) 自動設定関係のソースコードの整理(性能，使用法には変化なし)

### mrp-v06.2 ###############################################################################
mrp-v06.1からの変更点(mrp-v06.1 -> mrp-v06.2)
  a) Visual Studio 2008に対応
  b) レート等を"./Info/Verbose_log.csv"にまとめて吐き出す"VERBOSE_LOG_MODE"の追加
  c) クラスマップ等を出力する関数を"log.c"に移動
  d) ログ出力を強化("log.c")
	・予測誤差の推定符号量マップ(pm->cost[] + pm->subcost) -> ./Info/Rate_Map (encoder only)
	・予測器の振幅特性　-> ./Info/Amp_Chara
	  　 (* 横が水平方向の周波数，縦が垂直方向の周波数を示し，
		左上が直流，右下が最高周波数(fs/2)となる．
		また，出力値は"振幅利得[倍]×64"ですので64が1[倍]に対応します．)
	・特徴量と予測誤差の分散の関係 -> ./Info/Var_Upara
　e) ヘッダファイルを整理
　f) フラグのオンオフを"定義"or"未定義"から"1"or"0"に変更

### mrp-v06.1 ###############################################################################
2008/4/26
mrp-v06 からの変更点(mrp-v06 -> mrp-v06.1)
  a) ハフマン符号に関連する項目を削除 (ハフマン符号に関しては mrp-v06 を参照)
  b) 特徴量の重み付けをユークリッド距離から市街地距離へ変更．
  c) クラス数自動設定におけるブロックサイズ固定時のバグを修正．
  d) "COEF_PRECISION"をオプション"-P"で設定した場合,"MAX_COEF"が修正されなかった問題を修正．
  e) クラス数自動設定を用いた場合，まれにクラスを全て削除してしまうバグを修正．
  f) 特定のサイズの画像において,ブロック数が正しくカウントされない問題を修正．


その他の変更点
  a) ファイルの出力先を変更
	・MRPファイル(encoder) -> ./Encoded_File
	・PGMファイル(decoder) -> ./Decoded_File (通常は原画と比較後に削除するため残らない)
	・クラスマップ -> ./Info/Class_Map (encoder -> mrp_class : decoder -> pgm_class)
	・予測器 -> ./Info/Predictor       (encoder -> mrp_prd : decoder -> pgm_prd)
	・しきい値 -> ./Info/Threshold     (encoder -> mrp_th  : decoder -> pgm_th)
  b) 実行ファイル名を大文字に変更
　c) フォルダの初期化用シェルを追加 (init_directory.sh)
